# Functional Programming

## Dependencies

### JavaScript

Run below code in **js** folder:
```
npm install
```

### PureScript

Run below code in **ps** folder:
```
spago build
```

## Usage

### JavaScript

***Run below commands in js folder:***

To run **singleton**: 
```
npm run dev:singleton
```

To run **isEmptyArray**: 
```
npm run dev:isEmptyArray
```


To run **snoc**: 
```
npm run dev:snoc
```


To run **length**: 
```
npm run dev:length
```

### PureScript

To run **all** methods
```
cd ./ps
spago run
```
